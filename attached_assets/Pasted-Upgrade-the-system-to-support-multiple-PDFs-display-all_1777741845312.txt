Upgrade the system to support multiple PDFs, display all extracted questions, show their source page images, and visually highlight the selected question (approximate, stable approach).

========================
BACKEND
========================

1. Accept multiple PDF uploads

2. For each PDF:
- Extract text
- Extract questions from text
- Track approximate page number for each question
- Also generate a short snippet (first 8–12 words of the question)

3. Convert first 3–5 pages of each PDF into images (PNG)
- Use a stable library like pdf2pic or pdf-poppler

4. Return structured response:

{
  questions: [
    {
      question: string,
      snippet: string,
      source: {
        fileName: string,
        page: number
      }
    }
  ],
  pageImages: {
    [fileName]: string[]
  }
}

5. Ensure:
- System never crashes
- If one PDF fails, continue processing others
- Always return valid JSON

LEFT PANEL:
- Display corresponding page image:
  pageImages[fileName][page]

RIGHT PANEL:

"From: filename — Page X"
